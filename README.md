#### 2) Service Module for python3 --> https://pypi.python.org/pypi/service

## Notes
#### Working only on Linux.

## HOW TO RUN 

##### [+] In terminal (python3 pyservice.py COMMAND)
             [-]COMMAND : 
                    [--start] to start service.
                    [--stop] to stop service.
                    [--status] to check service status.
